Tools

これらのツールは HTML + CSS + JavaScript です。

DOM Event Monitor
マウス操作やキーボード入力により発生する DOM イベントをリアルタイムに監視します。
JavaScript Evaluater
テキストエリアに JavaScript のコードを入力し Enter を押すと実行されます。
JavaScript Object Tree
JavaScript のグローバルスコープにおける this オブジェクトのプロパティを列挙し、型や値を表示します。
Kana Roma Transformer
かな入力とローマ字入力を同時に行うことができます。
Morse Translator
アルファベットや数字のモールス符号化および複合を行います。
Unicode Table
Unicode の文字を byte 順に 256 字ずつ表示します。
URI Validator
RFC3986 に基づき、入力された文字列が正当な URI かどうかを判定します。
